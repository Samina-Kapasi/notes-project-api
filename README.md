# Notes API Project 📝

This is a backend Notes Management API built using FastAPI, SQLAlchemy, SQLite/MySQL, and JWT Authentication.

## Features

* User Signup API
* User Login API
* JWT Authentication
* Protected Routes
* Create Notes
* View Notes
* Update Notes
* Delete Notes
* Search Notes
* Filter Notes
* SQLAlchemy Database Integration

## Tech Stack

* FastAPI
* SQLAlchemy
* SQLite / MySQL
* JWT Authentication
* Pydantic
* Python

## Installation

Clone the repository:

```bash
git clone https://github.com/Samina-Kapasi/notes_project_api.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the server:

```bash
uvicorn main:app --reload
```
